# Coreflow

Coreflow es un ecosistema modular diseñado para desarrolladores, compuesto por microservicios y frontends con un enfoque fullstack.

## Introducción

Esta monorepo contiene múltiples microservicios construidos con diversas tecnologías para proporcionar flexibilidad y escalabilidad.

## Configuración Inicial

Consulte las guías en cada subproyecto para la configuración y uso detallados.
